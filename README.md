# Korean Slang Quiz  
  
This is a simple Python console quiz game about Korean slang abbreviations.  
  
## Features  
  
- Presents 3 questions about popular Korean slang abbreviations.  
- Asks the user to enter the full meaning of each abbreviation.  
- Counts and displays the number of correct and incorrect answers.  
  
## How to Use  
  
1. Run the script in a Python (3.x) environment.  
2. For each abbreviation shown, type the full phrase (no spaces).  
3. After 3 questions, your score will be displayed.  
  
## Example  
Welcome! I will give you 3 quiz questions about Korean slang. Try to get them right.
취존이 어떤 문장의 줄임말인가요?: 취향존중
Correct!
솔까말이 어떤 문장의 줄임말인가요?: 솔직히 까놓고 말해서
Correct!
케바케이 어떤 문장의 줄임말인가요?: 케이스바이케이스
Correct!
맞은 개수는 3개, 틀린 개수는 0개 입니다
